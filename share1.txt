Ah ok, that makes sense.  For HBSS those components would be the server hosting the HBSS solution and the agents deployed across the system. So can the server and agents be reached by ACAS, are the servers and agent locations access controlled on each system (this can be configured via GPO), who has permissions to make any changes to the server and agents in the HBSS management console; those are the types of attributes these requirements are looking for. Req 9 is more specific to the function on the HBSS and does it work as intended and does the real time scanning, so this would also have to be configured in the management console.
